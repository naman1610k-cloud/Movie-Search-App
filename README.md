#🎬 Movie Search App

A responsive movie search web application built using Javascript and the OMDb API.

## ✨ Features

-🔎 Movie Search with debounce
-📄 Movie details page
-📦 Pagination
-💀 Skeleton loading 
-💾 State persistence (restore search & page)
-🖼 Image fallback handling
-📱 Fully Responsive 

## Check live demo 🚀 
https://movie-search-app2007.netlify.app/

## 🛠 Tech Stack 

- HTML
- CSS
- JavaScript
- OMDb API
